# Usage Policy

## Project Status

**Lynknov** is an active in-development project owned and maintained by **Daffa Dhiyaulhaq Khadafi**.

This repository is published publicly for:

- portfolio visibility
- project progress documentation
- learning reference
- product presentation
- selective collaboration context

## Allowed

You may:

- view the repository contents
- read the source code and documentation for reference
- observe the product structure, workflow, and development approach
- discuss the project for educational, review, or collaboration purposes

## Not Allowed Without Explicit Written Permission

You may **not** do any of the following without prior written permission from the project owner:

- copy substantial parts of the source code into your own project
- reuse the product structure as a direct commercial implementation
- redistribute repository contents
- publish modified versions of the code or documentation
- create derivative products based substantially on this repository
- use the Lynknov name, branding, positioning, or documentation as your own
- use any part of the project for commercial, client, or production deployment

## Collaboration and Permission Requests

If you are interested in:

- collaborating on the project
- discussing educational use
- reviewing the system in more depth
- requesting permission to reuse limited parts of the repository

please contact the repository owner directly before using any material beyond normal public viewing.

## Ownership

Unless otherwise stated, all code, documentation, written materials, brand direction, product strategy, and related project assets in this repository remain under the ownership of the project owner.

Public visibility of this repository does **not** grant open-source usage rights.

## Future Changes

This policy may be updated in the future if the repository licensing or collaboration model changes.

Until such changes are explicitly published, this repository should be treated as **viewable but not reusable without permission**.
