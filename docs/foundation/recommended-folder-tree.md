## Prinsip Penempatan

- semua keputusan besar masuk `docs/`
- semua implementasi kode masuk `app/` dan `packages/`
- semua file visual mentah masuk `assets/`
- semua urusan SQL, migration, dan seed masuk `database/`
- semua eksperimen lama atau arah yang sudah ditinggalkan masuk `archive/`
