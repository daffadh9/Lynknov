# 1. Struktur utama sistem mingguanmu

Fokus hidupmu sekarang dibagi ke 5 area:

1. **Lynknov**
2. **Produk digital / monetisasi**
3. **Konten**
4. **Olahraga / kesehatan**
5. **Hiburan terkontrol**

Target besarnya:

* hidup tetap manusiawi
* ada progres nyata
* tidak kelelahan
* tidak tenggelam di dopamin cepat
* ada hasil mingguan

---

# 2. Aturan inti mingguan

Pegang ini dulu sebagai aturan dasar:

## Aturan wajib

* tidak buka game / hiburan sebelum blok kerja inti
* setiap hari harus ada **1 output**
* minimal **1 blok deep work** per hari kerja
* minimal **30 menit gerak tubuh**
* hiburan hanya di slot malam
* review harian 10 menit
* review mingguan 30–45 menit di hari Minggu

## Definisi output

Output itu contohnya:

* 1 bagian Lynknov selesai
* 1 draft produk digital jadi
* 1 konten dipublish
* 1 dokumen roadmap jadi
* 1 revisi penting selesai

Jadi bukan sekadar “tadi aku mikir”.

# 8. Aturan game dan hiburan yang lebih tegas

Biar tidak kabur lagi, pakai aturan ini:

## Rule hiburan

* tidak boleh game sebelum deep work selesai
* tidak boleh YouTube random saat break kerja
* hiburan hanya setelah jam malam
* kalau hari itu nol output, durasi hiburan dipotong
* maksimal hiburan harian: **1–2 jam**
* jangan main sampai lewat jam tidur

## Rule YouTube

Bedakan 2 jenis:

* **YouTube kerja/belajar**
* **YouTube hiburan**

Kalau kamu buka YouTube “belajar”, tapi lompat-lompat video tanpa implementasi, itu masuk hiburan, bukan belajar.

# 9. Mode hari: biar kamu tidak gampang menyerah

Gunakan sistem ini.

## Mode A — Strong Day

Kalau energimu bagus:

* kerjakan tugas berat
* maksimalkan pagi
* hasilkan output besar

## Mode B — Normal Day

Kalau energimu biasa:

* fokus target utama
* pendukung kalau sempat

## Mode C — Low Day

Kalau lagi turun:

* kerjakan target minimum
* tetap gerak tubuh
* jangan nol total

Contoh target minimum:

* buka file kerja 20 menit
* buat 1 outline
* rapikan 1 dokumen
* publish 1 konten ringan

---

# 10. Sistem anti-kebablasan dopamin

Ini penting banget buat kamu.

## Hal yang harus dilakukan

* workspace kerja harus langsung siap dipakai
* file target utama dipin
* notif tidak penting dimatikan
* shortcut game jangan di depan mata
* jangan buka short-form content di pagi hari
* break dipakai untuk minum, jalan, stretching, bukan selalu layar

## Prinsip utamanya

**Bikin kerja jadi gampang dimulai, bikin distraksi sedikit lebih susah diakses.**

# 13. 7 aturan founder pribadimu

Aku sarankan ini kamu tulis besar-besar.

1. Saya tidak memulai hari dengan hiburan.
2. Saya harus menghasilkan satu output setiap hari.
3. Saya mendahulukan progres kecil yang selesai daripada rencana besar yang tertunda.
4. Olahraga adalah bagian dari produktivitas saya.
5. Belajar tanpa implementasi bukan progres.
6. Hiburan adalah hadiah, bukan default.
7. Kalau saya jatuh dari ritme, saya kembali tanpa drama keesokan harinya.

---

# 14. Versi paling praktis untuk mulai besok

Kalau kamu mau start simpel dulu, mulai dengan 5 kewajiban ini:

## Besok wajib:

* 1 deep work block pagi
* 1 output nyata
* 30 menit gerak tubuh
* 1 blok kerja ringan / konten
* hiburan malam maksimal 1–2 jam

Itu saja dulu.

Kalau 5 hal ini kamu jaga 7 hari, hidupmu akan mulai terasa jauh lebih stabil.

# 15. Tantangan 7 hari awal

Biar lebih terasa, ini challenge kecilnya:

## Challenge 7 hari

Selama 7 hari:

* no hiburan sebelum deep work
* minimal 1 output per hari
* minimal 30 menit gerak tubuh
* isi daily review tiap malam
* Minggu malam isi weekly review

Kalau kamu bisa jalani ini, itu pondasi sangat kuat untuk challenge produk digitalmu.
