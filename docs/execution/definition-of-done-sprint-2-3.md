# Definition of Done — Sprint 2 & Sprint 3

## Sprint 2 dianggap selesai kalau:
- builder punya editing flow yang jelas
- hero/about/portfolio/cta bisa diedit
- portfolio items bisa dikelola
- page settings dasar tersedia
- dashboard menunjukkan status yang berguna
- founder bisa menyiapkan halaman dengan lebih percaya diri

## Sprint 3 dianggap selesai kalau:
- publish flow berjalan rapi
- halaman publik terasa layak untuk dipakai
- lead inbox dasar tersedia
- analytics summary dasar tersedia
- route protection dan validation sudah dicek
- founder bisa menjalankan full loop sendiri

## Belum termasuk done kalau:
- hanya UI terlihat bagus tapi flow masih membingungkan
- data tersimpan tapi sulit dikelola
- publish flow belum aman
- leads masuk tapi tidak bisa direview
- analytics ada tapi tidak actionable
