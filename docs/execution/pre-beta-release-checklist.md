# Pre-Beta Release Checklist

## Product
- onboarding flow clear
- dashboard useful
- builder usable
- publish flow stable
- public page credible
- lead inbox usable
- analytics summaries understandable

## Technical
- migrations clean
- env vars documented
- auth protection checked
- slug uniqueness checked
- no major console/runtime errors
- basic mobile layout checked

## QA
- empty states checked
- loading states checked
- form validation checked
- invalid routes checked
- unauthorized access checked
- publish/unpublish behavior checked

## Founder validation
- founder profile real data inserted
- founder page published
- founder CTA tested
- founder lead submitted
- founder reviewed analytics
- friction notes recorded
