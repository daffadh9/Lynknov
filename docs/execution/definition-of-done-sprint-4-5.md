# Definition of Done — Sprint 4 & Sprint 5

## Sprint 4 selesai jika:
- friction founder utama sudah berkurang
- dashboard lebih actionable
- builder lebih nyaman dipakai
- public page terasa lebih kredibel
- empty states dan feedback states lebih matang
- produk terasa lebih stabil untuk dipakai sendiri

## Sprint 5 selesai jika:
- produk terasa siap untuk trusted pre-beta
- baseline UX dan QA lebih kuat
- ada fondasi monetization ringan
- ada checklist release internal
- produk terasa lebih matang tanpa kehilangan fokus MVP

## Belum dianggap selesai jika:
- hanya visual makin cantik tapi flow tetap membingungkan
- agent menambah scope di luar kebutuhan
- monetization malah jadi distraksi besar
- produk belum cukup percaya diri untuk didemokan ke orang lain
