## Executive Summary

Roadmap 90 hari Lynknov harus berorientasi pada satu tujuan:
**meluncurkan MVP yang cukup tajam untuk dipakai, dibagikan, dan diuji sebagai business-ready profile system untuk service-led users.**

Roadmap ini dibagi ke empat fase:
1. strategic lock
2. product & design definition
3. build core system
4. launch & learn
