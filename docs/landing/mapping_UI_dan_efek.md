Ini yang bakal bikin Lynknov beda dari landing page SaaS biasa.

Tujuan section
Layout UI
Visual direction
Efek & motion
🔥 1. HERO (SECTION PALING KRUSIAL)
🎯 Tujuan
nangkep perhatian
bikin user “kena”
langsung positioning: trust
🧱 Layout UI
grid 2 kolom
kiri: headline + CTA
kanan: product preview (WAJIB kuat)
max-width lebar (7xl atau lebih)
vertical spacing lega
🎨 Visual Direction

Ini yang bikin beda:

Background
jangan flat hitam
pakai:
radial glow di belakang headline
subtle gradient top → transparent

Contoh rasa:

cahaya halus dari atas
ambience “soft spotlight”
Product Preview (KANAN)

Ini harus jadi “jualan utama”

Jangan:

box kosong
dummy card biasa

Harus:

layered UI
beberapa panel bertumpuk
ada:
profile
service
CTA
trust block

👉 feel: “ini produk nyata”

⚡ Efek & Motion
headline: fade-up + slight blur → clear
mockup kanan:
floating subtle (translateY kecil)
parallax ringan
CTA:
hover scale 1.02
glow:
very slow breathing effect
